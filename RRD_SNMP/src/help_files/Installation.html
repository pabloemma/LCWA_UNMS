Installation:

In order for LCWA_UNMS to work you need to have the following installed:

  1. python3.9 (not python3.10)

  2. a C++ compiler

  3. git

Once you have this you can do a

git clone https://github.com/pabloemma/LCWA_UNMS .

Next you need to do the following commands in a command line environment:

  1. pip3 install -U wxPython

  2. pip3 install -U pyyaml

  3. pip3 install -U pypubsub

  4. pip3 install -U requests

  5. pip3 install -U pandas

  6. pip3 install -U matplotlib
